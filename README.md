# microservice
Features
Basic operations: Addition, Subtraction, Multiplication, Division
Advanced operations:
- Exponentiation (a^b)
- Square root (√a)
- Modulo (a % b)

API Endpoints
Power (Exponentiation)
GET /power?base=2&exponent=3 → { "result": 8 }

Square Root
GET /sqrt?number=9 → { "result": 3 }

Modulo
GET /modulo?a=10&b=3 → { "result": 1 }
